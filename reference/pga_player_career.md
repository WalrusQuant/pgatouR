# Get player career data

Returns a player's career achievements including starts, cuts, wins,
finish distribution, and earnings.

## Usage

``` r
pga_player_career(player_id)
```

## Arguments

- player_id:

  Character. Player ID.

## Value

A tibble of career statistics.

## Examples

``` r
if (FALSE) { # \dontrun{
pga_player_career("52955")
} # }
```
