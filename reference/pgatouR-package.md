# pgatouR: R Client for the PGA Tour API

Access PGA Tour leaderboards, player stats, scorecards, shot tracking
data, tee times, FedExCup standings, odds, and more. Wraps the PGA Tour
GraphQL and REST APIs, returning tidy tibbles.
